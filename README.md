# Date Range Filtering, Color Highlighting and Reporting System

## Overview

This project is a C# Windows Forms application designed to manage records through date range filtering, visual color highlighting, and report generation. The application allows users to search records within specific dates, improve data visualization with color indicators, and generate detailed reports.

## Features

* Filter records by a selected date range.
* Highlight records using different colors based on predefined conditions.
* Generate reports from filtered data.
* User-friendly Windows Forms interface.
* Microsoft Access database integration.
* Fast and efficient data retrieval.

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* Microsoft Access Database (OLEDB)
* Crystal Reports

## How It Works

1. Users select a start date and an end date.
2. The application retrieves records that fall within the selected range.
3. Matching records are displayed in a DataGridView.
4. Rows can be automatically color-coded according to specific criteria.
5. Users can generate and view reports based on the filtered results.
